
## 🚀 Deployment Guide

Follow these steps to deploy the infrastructure using Terraform:

### 1. Clone the Repositor
git clone https://github.com/mohiuddin1221/Terraform.git


cd Terraform
##  Initialize Terraform
terraform init
##  Validate Configuration
terraform validate
##  Preview Changes
terraform plan
##  Apply Infrastructure
terraform apply
##  Cleanup Resources
terraform destroy
